Background

Fyform is a SaaS insurance workflow tool that not only helps you fill out applications, send applications direct to MGA, retrieves the correct forms for each line of business, and organizes all relevant client documents, it also performs commission calculations on every policy sold. This helps save massive amounts of time and effort usually spent on retrieving old files and repeatedly filling out the same information, so the agent can focus on selling.

Technology

Ruby on Rails was chosen as the framework for Fyform because of the out-of-box features and ]structure of the application. It allows me to quickly learn how to develop software. With Rails, I can utilize ActiveStorage, ActiveJob, and ActiveMailer in combination with Sidekiq to process jobs.
